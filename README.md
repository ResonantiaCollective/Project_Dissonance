# Project Dissonance — Structure Overview

Project Dissonance is the main development repository for the Resonantia Game Universe: a hybrid of rhythm combat, sacred cyber-mythology, hard techno, and codex-driven worldbuilding.

---

## 00_DOCUMENTATION
The Cyber-Sacred Codex:
- Living Bible (Lore)
- Technical Codices (Combat, Card, Player, Enemy, Trigger, Zone, AI Engines)
- Cinematics and narrative scrolls
- Branding references

---

## 01_GODOT_PROJECT
Active Godot prototype.

Contains:
- `/scenes` (combat, deck builder, UI, test arenas)
- `/scripts` (core, combat, cards, utils)
- `/resources` (audio, cards)
- `/addons`
  
This is the evolving gameplay core of Resonantia.

---

## 02_AUDIO
All game audio assets.

Includes:
- Operative-associated tracks
- Transmissions (lore broadcasts)
- SFX library
- Trigger data
- Masters and mixdowns

---

## 03_ART
Assets for visual development.

Contains:
- Card art pipeline (raw → processed → final)
- UI graphics
- Zones and environmental concepts
- House / Faction iconography
- General reference library

---

## 04_OPERATIVES
Profiles for every Operative in the universe.

Each Operative contains:
- Profile card + metadata  
- Sigils / Logos  
- Visual development  
- Audio kits / tracks  
- Lore scrolls and documentation  

---

## 05_PROGRESS
Development logs and tracking.

Contains:
- Daily logs  
- Weekly devlogs  
- Milestones  
- Progress tracker  

---

## 06_BACKUPS
Manual + automated backups for the entire project tree.

---

This structure supports multi-year, multi-disciplinary development and cross-team collaboration.

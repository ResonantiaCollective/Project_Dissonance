# ECHOFORGE COMBAT ANIMATION BLUEPRINT v1.0  
## Movement | Strikes | Aura | Encounter Presence

Defines animation needs for Echo//Forge as a hybrid Founder encounter.

---

# I. IDLE ANIMATION

Wolf–Oni shift every 4–8 seconds.  
Subtle coat sway.  
∞ loop rotates slowly.

---

# II. WALK & RUN CYCLE

Walk = disciplined, Oni-led  
Run = forward-heavy, Wolf-led

Transitions must reflect resonance mode.

---

# III. STRIKE ANIMATIONS

**Wolf Strike (Aggressive):**  
- wide arc  
- body twist  
- heavy follow-through  

**Oni Strike (Precise):**  
- straight line  
- minimal body movement  
- sharp halt  

**Dual Strike:**  
- start Oni → finish Wolf  
- or reverse, depending on mode  

---

# IV. SPECIAL — FOUNDER’S RESONANCE

- stance lowers  
- coat lifts from aura pressure  
- dual rings snap together  
- mask seam glows violet  
- full-body pose lock for 0.5s  

---

# V. REACTION ANIMATIONS

Perfect Guard → blue flash  
Wolf Rage → red burst  
Dual Stagger → ripple distortion

# OPERATIVE TEMPLATE

Use this structure for each Audio Operative:

- 00_PROFILE  → bio, role, House alignment, contact, agreements
- 01_SIGILS   → logos, marks, glyphs, personal seals
- 02_VISUALS  → images, covers, concept art, promo visuals
- 03_AUDIO    → stems, mixes, reference tracks
- 04_SAMPLES  → custom one-shots, loops, fx
- 05_DOCS     → briefs, agreements, tech notes
- 06_EXPORTS  → final packs to send/shared bundles

Duplicate this folder and rename it to OPERATIVE_<NAME>.

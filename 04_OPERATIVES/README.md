# Operative Template

This folder defines the required structure for creating new Operatives in the Resonantia Universe.

---

## Folder Structure

### 00_PROFILE
Operative Card, identity, metadata.

### 01_SIGILS
Sigils, runes, faction symbols, crests.

### 02_VISUALS
Character development:
- Art blueprint
- Turnarounds
- Animation references
- VFX packs
- Asset packs

### 03_AUDIO
Operative-specific audio assets:
- Themes
- SFX
- Voice references

### 05_DOCS
Lore scrolls, EPK, narrative files, worldbuilding notes.

---

## Naming Format
OPERATIVE_<NAME>_<ID>

Example:  
`OPERATIVE_MOLLYTUDE_ONI13`

---

## Versioning
Use clear version labels:
`v1.0`, `v1.1`, `v2.0`, etc.

Never delete older versions. Archive instead.
